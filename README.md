# Infrastructure Active Directory sous Windows Server 2019

## Présentation

Ce projet a pour objectif la mise en place d'une infrastructure Active Directory complète similaire à celle utilisée dans une PME. L'environnement comprend un contrôleur de domaine, des services DNS/DHCP, des stratégies de groupe (GPO) et plusieurs postes clients intégrés au domaine.

## Objectifs

* Déployer un contrôleur de domaine
* Mettre en place Active Directory
* Configurer DNS et DHCP
* Automatiser l'administration via PowerShell
* Gérer les utilisateurs et groupes
* Déployer des stratégies de sécurité

## Environnement

* Windows Server 2019
* Active Directory Domain Services
* DNS
* DHCP
* Group Policy Objects (GPO)
* PowerShell
* Clients Windows

## Réalisations

### Active Directory

* Création d'une forêt Active Directory
* Création et organisation des OU
* Gestion des utilisateurs et groupes
* Mise en place d'une structure par département

### Services réseau

* Configuration DNS
* Zones directes et inverses
* Configuration DHCP
* Réservations d'adresses IP

### Sécurité

* Politique de mots de passe
* Verrouillage des sessions
* Gestion centralisée des utilisateurs
* Déploiement de GPO

### Automatisation

* Scripts PowerShell
* Création d'utilisateurs via import CSV
* Gestion automatisée des groupes
